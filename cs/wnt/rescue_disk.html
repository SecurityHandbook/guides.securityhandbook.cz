Odvirování Windows pomocí záchranného disku

Pokud selže odstranění malware přes OS, případně nelze OS ani zavést, je nutné použít live OS s antimalware systémem na externím CD / USB disku. I k tomuto postupu je nezbytný přístup k internetu. Nejkomfortnějším způsobem je připojení síťovým kabelem. Níže naleznete postup pro Bitdefender Rescue CD a Kaspersky Rescue Disk – použitelné jsou obě varianty, i jejich kombinace.


Bitdefender Rescue CD

Stáhnutí a vytvoření záchranného disku:

  • Stáhněte si ISO soubor Bitdefender Rescue CD.
  • Vytvořte instalační médium z ISO souboru, pro USB doporučím program Rufus (návod zde).
  • Nabootujte z vytvořeného instalačního média (většinou klávesa F12 při BIOS/UEFI uvítací obrazovce).

Odstranění malware:

  • Zkontrolujte, zdali máte zvolenou možnost Start the Bitdefender Rescue CD in English, a stiskněte Enter.
  • Vyčkejte na dokončení všech příprav, dokud se neobjeví uvítací okno aplikace Bitdefender Rescue CD.
  • Odsouhlaste licenční podmínky aplikace.
  • Automaticky bude inicializována aktualizace a následný sken.
  • Po dokončení skenu se zobrazí nalezené hrozby. Rozklikněte nejvýše položený box pro globální volbu akce a zvolte možnost Disinfect.
  • V dolní liště klikněte na tlačítko Fix issues.
  • Zůstanou-li na seznamu položky, které nebylo možné dezinfikovat, zvolte pro ně možnost Delete nebo Rename a znovu klikněte na Fix issues.
  • Jakmile budou zneškodněny všechny nalezené hrozby, klikněte na Finish a následně Done.
  • Restartujte zařízení.

Úspěch
Váš OS by měl nyní být zbaven aktivního malware a schopný zavedení. Pokud ne, můžete zkusit ještě možnost uvedenou níže.




Kaspersky Rescue Disk

Stáhnutí a vytvoření záchranného disku:

  • Navštivte stránky Kaspersky a stáhněte si ISO soubor Kaspersky Rescue Disk.
  • Vytvořte instalační médium z ISO souboru, pro USB doporučím program Rufus (návod zde).
  • Nabootujte z vytvořeného instalačního média (většinou klávesa F12 při BIOS/UEFI uvítací obrazovce).

Odstranění malware:

  • Klávesou 1 odsouhlaste licenční podmínky.
  • Zkontrolujte, zdali máte zvolenou možnost Kaspersky Rescue Disk. Graphic Mode, a stiskněte Enter.
  • Vyčkejte na dokončení všech příprav, dokud se neobjeví okno aplikace Kaspersky Rescue Disk.
  • V hlavním panelu se přesuňte do záložky My Update Center a zvolte možnost Start update.
  • Po dokončení aktualizace v horním panelu klikněte na Settings.
  • Zvyšte Security level na úroveň High a klikněte na OK.
  • V hlavním panelu se přesuňte do záložky Objects Scan.
  • Zatrhněte všechny dostupné disky pro jejich kontrolu a zahajte sken tlačítkem Start Objects Scan.
  • Po dokončení skenu se v pravém dolním rohu zobrazí vyskakovací okno s nalezenými hrozbami. Zatrhněte možnost Apply to all objects a následně zvolte doporučovanou akci.
  • Postup opakujte pro všechny druhy nalezených hrozeb.
  • Aplikaci zavřete a restartujte zařízení.

Úspěch
Váš OS by měl nyní být zbaven aktivního malware a schopný zavedení.
